How does JavaScript work

Step: 1

At the first stage of JS engine is Perser. Inside JS engine Parser received the code and check syntactic error line by line. If the parser found an error its throw various kind of error and stop execution of the code.

Step: 2

After checking all code by Parser if no error found in the code it creates the data Structure that is AST (Abstract Syntax Tree).

Step: 3

After Creating AST by parser, JS engine converts JS code into machine language.

Step: 4

After converting the code into machine language the code is sent to the system for execution and finally the code is executed by system/engine.

Difference Between Local Storage and Session Storage

Local Storage

1. Its store data long time although you close a tab or close the browser

2. Its lasting up to the data is not clearing manually.

Sesseion Storage

1. Its store data for a short time after close a tab or window the session storage data is automatically clear.

2. Its lasting when the tab or browser closed.

3. When open a page in a new tab or window creates a new session with the value of the top-level browsing context, which differs from how session cookies work
